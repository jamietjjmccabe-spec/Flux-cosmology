# Flux/MIP Missing History RC1 Package

This package contains the RC1 paper draft and supporting benchmark artifacts for the Missing History formulation of galactic residual gravity.

## Contents

- `paper/flux_missing_history_rc1.md` - markdown manuscript.
- `paper/flux_missing_history_rc1.tex` - LaTeX manuscript.
- `paper/flux_missing_history_rc1.pdf` - compiled PDF manuscript.
- `figures/` - paper figures.
- `data/` - benchmark CSV/JSON outputs and crossmatch catalogs.
- `scripts/` - pertinent Python scripts used in the empirical SFH and radial benchmark pipeline.

## RC1 result stack

1. Empirical z0MGS history signal: pass.
2. Flux radial gate: pass.
3. Unified bounded radial model: pass.
4. Wide-grid bounded-map stability: pass.

## Notes

The RC1 benchmark is provisional and intended for falsifiable follow-up. The next tests are full-SPARC radial validation, independent SFH catalogs, and cluster-scale lensing checks.
